**Pic-A-Day** is a visual diary Android app built to capture and organize your daily memories through photos and notes in an intuitive calendar interface.

## ✨ Features
* **Visual Calendar:** Preview your memories directly on the calendar grid.
* **Photo Journaling:** Attach multiple images and detailed notes to any specific date.
* **Cloud Sync:** Securely store and access your memories anywhere using Firebase.
* **Private Access:** Secure authentication system to keep your diary personal.

## 🛠️ Tech Stack
* **UI:** Jetpack Compose with Material 3.
* **Language:** Kotlin.
* **Backend:** * **Firebase Auth:** User management.
    * **Cloud Firestore:** NoSQL database for notes and metadata.
    * **Cloud Storage:** Image hosting.
* **Image Loading:** Coil.
* **Navigation:** Compose Navigation.

## 🏗️ Architecture
The project follows **MVVM (Model-View-ViewModel)** pattern and **Clean Architecture** principles to ensure scalability and maintainability.
